# Duolingo 风格重设计

## 概述

将单词练习小程序从基础信息型界面升级为 Duolingo 风格的游戏化学习体验。纯前端改造，不接入后端，所有数据继续使用 wx.Storage 本地存储。

## 设计决策

- 视觉风格：Duolingo 明快配色 + 大圆角卡片 + 阴影
- 游戏化程度：完整（XP、打卡、每日任务、成就）
- 练习交互：选中后自动判题 + 自动推进下一题
- 动效级别：丰富（撒花、震动、粒子、XP飞入）

## 1. 设计系统

### 颜色

| Token | 值 | 用途 |
|-------|------|------|
| primary | #58CC02 | 正确/完成/主按钮 |
| accent | #1CB0F6 | 链接/交互/进度 |
| error | #FF4B4B | 错误/生命值 |
| gold | #FFC800 | XP/奖励/金币 |
| bg | #F7F8FA | 页面背景 |
| card | #FFFFFF | 卡片背景 |
| text | #3C3C3C | 主文字 |
| textSecondary | #777777 | 次要文字 |
| border | #E5E5E5 | 边框 |

### 圆角与阴影

- 卡片：border-radius 20rpx, box-shadow 0 4rpx 16rpx rgba(0,0,0,0.08)
- 按钮：border-radius 16rpx, 阴影更重
- 选项：border-radius 16rpx

### 按钮

- 主按钮：绿底白字，高度 96rpx，按下 scale(0.97) 反馈
- 次按钮：白底灰边，同尺寸
- 选项按钮：白底，选中后变色（正确绿/错误红）

### 字体

- 标题：36-48rpx, font-weight 700
- 正文：28-32rpx, font-weight 400
- 辅助：24rpx, 次要文字色

## 2. 首页 — 游戏化仪表盘

### 布局

1. **顶部统计条**：🔥连续打卡天数 | ⭐总XP | 📖今日已学词数
2. **今日进度环**：Canvas 绘制圆环进度，显示"X/Y 题完成"，目标默认10题
3. **快捷入口网格**：2x2 网格，每个入口带动效卡片
   - 练习 📝
   - 复习 📖（有待复习数量角标）
   - 测试 🎯
   - 教材 📚
4. **复习提醒条**：有待复习错题时显示，可点击进入复习

### 数据

- 从 storage 读取：getUserStats(), getDueReviewItems()
- 今日进度：统计当天 practice_records 数量
- XP：累计存储在 user_stats.xp

## 3. 练习页 — 自动推进卡片

### 交互流程

1. 显示题目卡片（从右侧滑入）
2. 用户选择选项
3. 选中后 300ms 自动判题：
   - 答对：选项变绿 + ✅ 图标 + 粒子撒花 + "+10 XP" 飞入顶部
   - 答错：选项变红 + 正确答案高亮 + 卡片震动(translateX 抖动)
4. 600ms 后自动滑入下一题（当前卡片左滑出，新卡片右滑入）
5. 最后一题完成后跳转结果页

### 拼写题

- 输入框 + 确认按钮
- 确认后自动判题，同上动效
- 打乱字母作为提示

### 配对题

- 左右两列，点击配对
- 配对正确：两项同时变绿
- 全部配对完成：撒花 + 自动进入下一题

### 动效实现

- 粒子撒花：Canvas 绘制随机颜色圆形粒子，自由落体 + 旋转
- 卡片滑入/出：CSS animation transform translateX
- 卡片震动：CSS animation translateX 抖动 3 次
- XP 飞入：绝对定位元素从题目位置飞向顶部 XP 计数器
- 进度条：CSS transition width

### 选项样式

- 默认：白底灰边
- 悬停/按下：浅蓝底
- 选中正确：绿底 #58CC02，白字，✅
- 选中错误：红底 #FF4B4B，白字，❌
- 正确答案（答错时）：绿底高亮

## 4. 游戏化系统

### XP 经验值

- 答对一题：+10 XP
- 答错一题：+2 XP（参与奖）
- 完成一整轮：+50 XP 奖励
- 存储在 user_stats.totalXP

### 连续打卡

- 每日完成至少 1 轮练习（≥5 题）即算打卡
- 🔥 图标 + 天数显示
- 打卡日期记录在 user_stats.streakDates[]

### 每日任务

存储在 user_stats.dailyQuests，每天重置：

1. 完成 10 道题
2. 正确率达到 80%
3. 复习 5 个错题

完成任务额外 +20 XP。

### 成就徽章（预留）

定义成就列表，一期只做数据结构和展示，不做强提醒：

- 初学者：完成第一次练习
- 坚持者：连续打卡 7 天
- 词汇达人：累计学习 100 词
- 满分王：单次练习满分

## 5. 结果页 — 成绩单

### 布局

1. **顶部反馈**：根据分数显示不同 emoji 和文案
   - 100 分：🎉 太完美了!
   - ≥80：👏 太棒了!
   - ≥60：💪 继续加油!
   - <60：📖 再试一次!
2. **分数展示**：大字体分数 + 答对数/总数
3. **XP 奖励**：显示本轮获得的 XP，有飞入动效
4. **错题列表**：卡片式，显示单词 + 释义 + 题型
5. **操作按钮**：再练一次 / 返回首页
