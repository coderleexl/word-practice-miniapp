# 单词练习模拟试题小程序

这是一个面向英语单词学习与模拟测验的微信小程序立项包。项目目标是支持导入教材词表，并围绕教材生成类似多邻国的轻量互动练习，包括选择、听写、拼写、配对、完形和模拟试题。

## 当前阶段

立项阶段。

## 一期目标

- 支持手动录入和批量导入教材词表
- 支持按教材、单元、难度组织单词
- 支持多种练习题型
- 支持错题本和复习队列
- 支持模拟试题生成和答题记录
- 支持微信小程序端完成完整学习闭环
